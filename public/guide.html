FUNCTION FractionalKnapsack(items, W, strategy):

    // ‡∏Ç‡∏±‡πâ‡∏ô‡∏ï‡∏≠‡∏ô‡∏ó‡∏µ‡πà 2: ‡πÄ‡∏£‡∏µ‡∏¢‡∏á‡∏•‡∏≥‡∏î‡∏±‡∏ö‡∏ï‡∏≤‡∏°‡πÄ‡∏Å‡∏ì‡∏ë‡πå‡∏ó‡∏µ‡πà‡πÄ‡∏•‡∏∑‡∏≠‡∏Å
    IF strategy = "ratio" THEN

    ELSE IF strategy = "value" THEN

    ELSE IF strategy = "weight" THEN

    // ‡∏Ç‡∏±‡πâ‡∏ô‡∏ï‡∏≠‡∏ô‡∏ó‡∏µ‡πà 3: ‡∏´‡∏¢‡∏¥‡∏ö‡∏Ç‡∏≠‡∏á‡πÉ‡∏™‡πà‡πÄ‡∏õ‡πâ
    remainingCapacity ‚Üê W

        IF item.weight ‚â§ remainingCapacity THEN
            // ‡∏´‡∏¢‡∏¥‡∏ö‡∏ó‡∏±‡πâ‡∏á‡∏ä‡∏¥‡πâ‡∏ô (100%)
            fraction ‚Üê 1.0

            fraction ‚Üê remainingCapacity / item.weight

            valueTaken ‚Üê item.value √ó fraction

        // ‡∏≠‡∏±‡∏õ‡πÄ‡∏î‡∏ï‡∏Ñ‡πà‡∏≤
        totalValue ‚Üê totalValue + valueTaken

    RETURN totalValue, selectedItems

    // ‡∏Ç‡∏±‡πâ‡∏ô‡∏ï‡∏≠‡∏ô‡∏ó‡∏µ‡πà 1: ‡∏™‡∏£‡πâ‡∏≤‡∏á‡∏ï‡∏≤‡∏£‡∏≤‡∏á DP ‡∏Ç‡∏ô‡∏≤‡∏î (n+1) √ó (W+1)

        FOR w ‚Üê 0 TO W:                    // ‡∏ß‡∏ô‡πÅ‡∏ï‡πà‡∏•‡∏∞‡∏Ñ‡∏ß‡∏≤‡∏°‡∏à‡∏∏

                dp[i][w] ‚Üê dp[i-1][w]

                notTake ‚Üê dp[i-1][w]              // ‡πÑ‡∏°‡πà‡πÄ‡∏•‡∏∑‡∏≠‡∏Å‡∏Ç‡∏≠‡∏á‡∏ä‡∏¥‡πâ‡∏ô‡∏ô‡∏µ‡πâ
                take ‚Üê dp[i-1][w - items[i].weight] + items[i].value

                dp[i][w] ‚Üê MAX(notTake, take)

    maxValue ‚Üê dp[n][W]

    // ‡∏Ç‡∏±‡πâ‡∏ô‡∏ï‡∏≠‡∏ô‡∏ó‡∏µ‡πà 4: Traceback ‡∏´‡∏≤‡∏£‡∏≤‡∏¢‡∏Å‡∏≤‡∏£‡∏ó‡∏µ‡πà‡πÄ‡∏•‡∏∑‡∏≠‡∏Å
    selectedItems ‚Üê []
